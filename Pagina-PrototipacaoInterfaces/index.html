1. O que é o Balsamiq Wireframes? Qual estilo de prototipação ele utiliza?

O Balsamiq Wireframes é uma ferramentas de prototipação de interfaces de forma rápida e fácil para aplicativos web e móveis. Ele utiliza utiliza a prototipação de baixa fidelidade, não focando nos minímos detalhes.

2. O que é o figma? Qual estilo de prototipação ele utiliza?

O Figma é uma ferramenta de design de interface e prototipação colaborativa baseada na web. Ele utiliza a prototipaçãode alta fidelidade, podem ser muito detalhados e realistas, se aproximando bastante do produto final.
